Installation Guide

Follow the steps below to set up the Licensing System on your local machine or server:

Prerequisites

  • PHP 7.4+ installed
  • MySQL database
  • Composer (for PHP dependencies)

Installation Steps

  1. Clone the repository: git clone https://github.com/your-repo/licensing-system.git
  2. Navigate to the project folder: cd licensing-system
  3. Install Composer dependencies: composer install
  4. Copy the .env.example file to .env and configure your database and API keys.
  5. Set up the database using the provided schema.
  6. Run the project using your local web server (e.g., Apache, Nginx).
  7. Visit http://localhost in your browser to access the application.
